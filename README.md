# WebApp-Internship-
Can try the demo at https://web-app-internship.netlify.app/

## Install the frontend

```bash
git clone git@github.com:AditiRout/WebApp-Internship-.git
cd client
npm install
npm start
```

## Install the backend

```bash
git clone git@github.com:AditiRout/WebApp-Internship-.git
cd server
npm install
npm start
```
